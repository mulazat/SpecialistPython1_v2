## "Трехзначное ли"

### Задание

Проверьте, является ли данное число трехзначным.

### Формат входных данных

Дано целое число.

### Формат выходных данных

Вывести "Да", если данное число трехзначное и "Нет" в противоположном случае.

### Решение задачи

a=int(input("Введите а\n"))
if 100<=a<=999:
    print ("Это 3х знач")
else:
    print ("Не 3х знач")
